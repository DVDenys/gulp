2. Папка app, создаем папку sass, в ней главный style.css, туда импортим остальные файлы, пример импорта @import 'wrapper';
3. Пример назвы файла _wrapper.scss;

Порядок импортов для остальных файлов :
1. Подключаем в html
2. Для шрифтов, изображений, видео и скрипта свои папки с названиями: fonts, img, video, script соответственно;
